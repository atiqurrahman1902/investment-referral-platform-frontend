# Investment Referral Platform

A full-stack MERN (MongoDB, Express.js, React.js, Node.js) application that allows users to register, login, invest, and earn referral rewards. The application includes JWT authentication, MongoDB Atlas integration, React Dashboard, Referral Income Calculation, and Cron Job implementation.

---

# Project Overview

The Investment Referral Platform is designed to simulate a real-world investment and referral management system.

Users can:

- Register an account
- Login securely
- Invest money
- View Dashboard
- Earn referral income
- Track wallet balance
- View referral details

The project follows a clean folder structure with separation of concerns using Controllers, Routes, Models, Middleware, Services, and React Components.

# API Documentation

## Register User

### POST

```
/api/auth/register
```

Request

```json
{
"name":"Atiq",
"email":"atiq@gmail.com",
"mobile":"9876543210",
"password":"123456"
}
```

Response

```json
{
"message":"Registration successful"
}
```

## Login

### POST

```
/api/auth/login
```

Request

# Assumptions Made During Development

- Each user has a unique email.
- Passwords are encrypted before storing.
- JWT authentication is required for protected APIs.
- MongoDB Atlas is used as the cloud database.
- Referral income is generated when a referred user makes an investment.
- Wallet balance is automatically updated after referral income is generated.
- Users must be authenticated before making investments.
